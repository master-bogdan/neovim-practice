# Practice Drills

Real source files to open in Neovim and edit as exercises.

## How to use

1. Open a drill file in nvim
2. Read the OBJECTIVE comment at the top
3. Perform the edits using only the keyboard
4. Check your result against the EXPECTED comment at the bottom
5. Undo all changes with `u` (repeat) and try again faster

## Timing targets

Each drill has three targets:
- **Expert** — you've internalized this; it's muscle memory
- **Proficient** — solid, no hesitation on which keys to press
- **Learning** — still thinking about the motions

## Directories

- `01-motions/` — cursor movement, jumps, Flash precision
- `02-editing/` — operators, text objects, surround, registers, yanky clipboard
- `03-search-replace/` — find, substitute, cgn, global commands
- `04-navigation/` — file switching, buffers, windows, projects
- `05-lsp/` — go-to-definition, diagnostics, code actions
- `06-refactoring/` — extract, inline, rename, restructure
- `07-multifile/` — cross-file edits, batch operations, quickfix workflows
- `08-git/` — conflict resolution, hunk staging, blame, DiffView
